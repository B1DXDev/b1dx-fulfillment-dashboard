🖥
Layer 01
Frontend (Next.js 15)
  • React Query useMutation/useQuery
  • Loading state + skeleton UI
  • Human-in-the-loop verify before commit
  • ใช้ omsRequest() เดิม → Bearer + Tenant ฟรี

Backend (.NET 10 + Mediator)
  • Mediator Command/Query (CQRS)
  • IAiTextService interface (vendor-agnostic)

  • Audit log + tenant isolation บังคับทุก call

Layer 03
AI Provider + Storage
  • Claude API (Anthropic) — primary
  • Prompt caching native (Anthropic)
  • PostgreSQL + pgvector (embedding)
  • ai_usage_log table (billing/quota)

Roadmap — 3 Phase

Validate value ก่อน scale · Quick Win → Expand → Enterprise
🗓 PLAN
▸ Phase 1 · Quick Win
PIM Auto-description
~2 สัปดาห์ · 1 BE + 1 FE

  • FE: ปุ่ม "✨ Generate ด้วย AI" ใน Product Edit

  • Audit log + tenant isolation

▸ Phase 2 · Expand

  • BE: CreateOrderCommand + ParseOrderText
  • BE: pgvector setup + embedding pipeline

ลดเวลา CS create order 40-60%
▸ Phase 3 · Scale
Bulk Ops + Internal Chatbot
~6-8 สัปดาห์ · 2 BE + 1 FE
  • Sync Worker: Batch API integration
  • Bulk auto-tag + bulk translation
  • Order issue summarization (nightly)
  • RAG chatbot สำหรับพนักงาน fulfillment
Automation ระดับ enterprise · scale พร้อม

ความเสี่ยง & การจัดการ

6 ความเสี่ยงหลัก · ทุกข้อมี mitigation ชัดเจน — ไม่มี blocker
⚠ RISKS
!
Cost บานปลาย
AI token cost คุมยาก ถ้ามี user abuse หรือ infinite loop
MITIGATERate limit ต่อ tenant + budget alert + prompt cache + monthly cap
!
Data Privacy
ส่งข้อมูลลูกค้า/order ไปยัง 3rd party (Anthropic/OpenAI)
MITIGATEAnonymize PII ก่อนส่ง · Zero-retention policy · self-host ถ้า sensitive
⚠
AI Hallucination
AI สร้างข้อมูลที่ไม่มีจริง (สินค้าผิด, ราคาเพี้ยน)
MITIGATEHuman-in-the-loop verify ทุกครั้ง · ห้าม AI ตั้งราคา/SKU เอง
⚠
Latency
AI call ใช้เวลา 2-5s — ช้ากว่า REST API ปกติ
MITIGATEStreaming response · skeleton UI · Batch สำหรับงานไม่ realtime
⚠
Tenant Isolation
Prompt cache อาจรั่ว tenant context ข้ามกัน
MITIGATECache key รวม tenant_id เสมอ · ใช้ ICurrentTenant · audit ทุก call
✓
Vendor Lock-in
ผูกกับ Anthropic/OpenAI · เปลี่ยน vendor ยาก
MITIGATEInterface IAiTextService abstract · เปลี่ยน implementation ได้
