BeOne OMS — Full Feature Overview

Order Management System สำหรับ Fulfillment & Warehouse Management

1ระบบ Login & สิทธิ์
  • FF Member — จัดการ Brand/Shop
  • Brand Admin — ดูแลสินค้า Brand
  • FF Manager — จัดการ Warehouse
  • WH Admin / WH Worker
  • System Admin
  • รองรับ Mobile / Tablet / Desktop
2Master Data
  • Shop / Brand / SKU Item
  • Virtual Set (Bundle)
  • Promotion Engine
  • Shipping Courier
  • Safety Stock / Dummy Stock
  • FIFO / FEFO Config
3Order Input
  • API/Webhook — Shopee, Lazada, TikTok, LINE, Lnwshop
  • POS (Point of Sale)
  • Import Excel
  • Sale Page (ลูกค้ากรอกเอง)
  • Service Policy (Instant / Express)
4Order Preparation
  • แตก Virtual Bundle → SKU จริงอัตโนมัติ
  • ตรวจสอบ Promotion อัตโนมัติ
  • จัดลำดับ Order ตามวัน/เวลา
  • กระจายงาน Pick ให้พนักงาน
  • พิมพ์ Order Ticket
  • Skip ถ้ายังไม่ชำระ (ยกเว้น COD)
5WMS Operations
  • Inbound — รับสินค้าเข้าคลัง
  • Pick (Single / Batch / Team)
  • Pack + บันทึกวิดีโอ
  • Ship + ชั่งน้ำหนัก + Label
  • Return + Cycle Count
  • Lot Control (FIFO / FEFO)
6Order Status Flow
  • 100 รอจับคู่สินค้า
  • 400 ยืนยันแล้ว → 450 กำลังแพ็ค
  • 490 รอขนส่ง → 500 ส่งแล้ว
  • 700 จัดส่งสำเร็จ
  • 600–650 ตีกลับ (Return flow)
  • 800 ยกเลิก
7การเงิน & เอกสาร
  • Tax Invoice / ใบเสร็จรับเงิน
  • Credit Note / Debit Note
  • COD Report & Reimbursement
  • Subscription & Billing
  • Transfer / Finance Report
8จัดการสต๊อก
  • Real-time stock sync ทุก platform
  • คำนวณ Available Stock อัตโนมัติ
  • Location / Bin management
  • Replenishment
  • Barcode / QR scan
9Report & Monitor
  • Sales Report (Brand / Shop / SKU)
  • Stock Balance / Movement
  • Monitor Order / Inbound / Return
  • Worker Performance Dashboard
  • Export Excel
10Activity Traceability
  • Log ทุกกิจกรรมของพนักงาน
  • บันทึกวิดีโอตอน Pack สินค้า
  • ตรวจสอบย้อนหลังได้ทุก step
11เชื่อมต่อ Platform
  • Shopee / Lazada / TikTok
  • LINE Myshop / Lnwshop
  • WooCommerce / Shopify
12ระบบ Promotion
  • ซื้อ A แถม X
  • ซื้อ A และ B แถม X
  • ซื้อ A หรือ B และ C หรือ D แถม X
  • กำหนดวันเริ่ม-สิ้นสุด Promotion
